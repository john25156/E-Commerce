How to Run the E-commerce Site

Products | Cart | Login | Register

Setup and Run Instructions

  1. Open your terminal or command prompt.
  2. Navigate to the project directory: 
    cd c:/Users/Admin/OneDrive/Desktop/Intern/ecommerce-site
  3. Install dependencies: 
    npm install
  4. Ensure MongoDB is running locally on your machine (default port 27017).
  5. Start the server: 
    npm start
    or for development with auto-reload: 
    npm run dev
  6. Open your browser and go to: 
    http://localhost:3000

Troubleshooting

  • npm error enoent: Make sure you run npm commands inside the ecommerce-site directory where package.json is located.
  • MongoDB connection failure: Ensure MongoDB service is running locally. You can start it using your system's service manager or by running mongod in a separate terminal.
  • Port conflicts: If port 3000 is in use, set a different port by setting the environment variable PORT before starting the server, e.g., PORT=4000 npm start.
